




# StartApp 

simple boilerplate ready for development 

## Notes
- Currently supported for  frameworks:  FastAPi, Flask, Django


### Installation ###

```sh
 $ sudo pip3 install startapp
```


### Guide

```bash

startapp 

for fastapi:


for framework flask:

source .venv/bin/activate
export FLASK_APP=app.app

export settings=dev

flask run

```

# Supported OS
Linux, MacOS
